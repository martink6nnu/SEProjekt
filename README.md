# SEProjekt
Veebileht tutvustab veebilehe loojate koduloomi piltide abil. Avalehel saab valida, millise looma kohta kasutaja rohkem pilte soovib näha.
Tiimiliikmed on Sander Valt, Martin Kõnnussaar ja Artur Kreegipuu.

Tööjaotus

Iga liige teeb vahelehe enda kodulooma kohta ise. Vahelehtede stiili ühtlustamine toimub kui vahelehed on kõigil rühmaliikmetel valmis tehtud. Lõpuks on kõigil vahelehtedel ühine CSS fail.

Avalehe loomine koos CSS failiga toimub kõigi liikmete koostöös kasutades Discordi.

Kodulehe paneb üles Martin.
